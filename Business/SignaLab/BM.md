# SignaLab 비즈니스 모델

## 가격 정책

| 티어 | 월간 | 연간 | 기능 |
|------|------|------|------|
| **Free** | $0 | - | 3개 알림/일, 관심종목 5개, 24시간 지연 |
| **Basic** | $6.99 | $59.99 (28% 할인) | 무제한 알림, 관심종목 20개, 한글 요약 |
| **Pro** | $9.99 | $83.99 (30% 할인) | + 내부자/기관 데이터, 락업 캘린더, API |

## 경쟁 해자 (Moat)

| 요소 | 설명 |
|------|------|
| 데이터 결합 | 8-K + SI + LLM 조합은 복제 비용 높음 |
| 한국어 현지화 | 영어권 서비스 진입 장벽 |
| 네트워크 효과 | 커뮤니티 기능으로 락인 |
| 히스토리컬 데이터 | 시간 경과에 따른 백테스팅 가치 증가 |

## 리스크 & 대응

| 리스크 | 대응 |
|--------|------|
| LLM 비용 급증 | 캐싱, 프롬프트 최적화, 로컬 LLM 폴백 |
| FINRA/SEC API 변경 | 다중 소스, 스크래핑 백업 |
| 경쟁사 진입 | 한국어 + 커뮤니티 차별화 |
| 법적 이슈 | "투자 조언 아님" 면책 조항 |

## GTM 전략

### Phase 1: 런칭

- 한국 투자 커뮤니티 (Reddit, DC인사이드, 네이버 카페)
- Product Hunt 런칭
- 앱스토어 ASO 최적화

### Phase 2: 성장

- 콘텐츠 마케팅 (숏스퀴즈 사례 연구)
- YouTube/블로그 튜토리얼
- 인플루언서 파트너십 (재테크 유튜버)

### Phase 3: 확장

- 영어 버전 출시
- 추가 시장 (일본, 대만)
- B2B API 제공
