
# 김이사님 (SMBPieGlobal)
- DB 현금 관련 테이블 코인 관련으로 변경할 것
- 일반적으로 블록체인 내에서 코인/토큰 전송 시 실패 사례는 크게 두 가지
	- 블록체인의 시스템 부하
	- 주소를 잘못 입력할 경우
- 일단은 DT IN에서 USDT만 사용

### 회원가입
- 트론 지갑주소 생성 기능이 빠져 있어서 추가함
- 따라서 앱에서 회원가입은 'register_v3'로 사용해야 함

### 홈 스크린
- 여러가지 코인 잔액이 나와야 함
		- pol bnb trx smbs smb usdt-pol usdt-bnb usdt-trx
- 코인 정보 / 토탈 자산 세부 내역을 탭으로 변환하여 복잡해지지 않도록

### DT IN 스크린
- 가스비 확인 UI가 없어 UX가 좋지 않음
- 회사로 입금해야 될 주소는 'SMB Wallet Address'라고 되어있는데 DT IN Address라고 변경하는 것이 좋지 않을까?
- 코인 잔액 조회 API (themeController.getCoinBalance)
- 코인 입금 검증 (themeController.usdtDepositTxidConfirmRequest)
- 코인 입금 내역 테이블 (tbl_coin_deposit_request)
