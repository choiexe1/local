# 모델 평가

- 컴퓨터 비전 모델 학습 후 프로젝트의 목표를 충족하는지 확인하는 단계

# 성능 메트릭

- 모델의 성능을 직관적인 수치로 이해하는 데 도움이 된다.
- 성능 측정을 위해 다양한 메트릭이 사용된다.

몇 가지 주요 메트릭은 다음과 같다.

### 신뢰도 (Confidence)

- 신뢰도 점수는 객체가 특정 클래스에 속할 것이라는 모델의 확신을 나타낸다.
- 범위는 0에서 1 사이이며, 점수가 높을수록 신뢰도가 높다.

### IoU (Intersection over Union)
- IoU는 예측된 바운딩 박스가 실제 바운딩 박스와 얼마나 겹치는지를 측정하는 객체 감지의 지표
- IoU의 값은 0에서 1 사이이며, 1은 완벽한 일치를 나타낸다.
- 이 지표는 예측된 경계가 실제 객체 경계와 얼마나 일치하는지 측정하기 때문에 중요하다.

### mAP (Mean Average Precision)
- 평균 정밀도는 객체 감지 모델의 성능을 측정하는 방법이다.
- 각 객체 클래스를 감지하는 정밀도 점수를 평균화하여 모델이 객체를 얼마나 정확하게 식별하고 분류할 수 있는지 확인할 수 있는 지표

|             |                                                                                                                                                                 |
| ----------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| mAP\@.5     | 0.5의 단일 IoU 임계값에서 평균 정확도를 측정한다. 모델이 느슨한 정확도 요구 사항으로 객체를 찾을 수 있는지 확인한다.<br><br>완벽한 탐지보단 객체가 대략적으로 올바른 위치에 있는지 여부에 중점을 둔다. 모델이 일반적으로 객체를 식별하는 데 능숙한지 확인하는 데 도움이 됨 |
| mAP\@.5:.95 | 0.05 간격으로 0.5에서 0.95까지의 여러 IoU 임계값에서 계산된 mAP 값의 평균, 이 지표는 더 자세하고 엄격하다.<br><br>정확한 객체 탐지를 하는 모델인지 평가할 수 있는 지표                                                    |

## mAP를 높이는 방법 (모델 정확도 향상 방법)
1. 학습 데이터의 품질
2. 최적화 알고리즘
3. 어노테이션 작업 개선

# 가변 이미지 크기
- 일반적으로 YOLO는 모든 이미지를 정사각형 (640x640)으로 강제 리사이징하여 처리한다.
- `rect=true`를 사용하면 이미지의 원본 비율을 유지하면서 처리한다.
- 이 기능을 사용하면 다음과 같은 이점을 얻을 수 있다.

1. 왜곡 방지: 이미지 비율을 유지한 채 리사이징 후 처리하므로, 다양한 크기의 이미지정확도를 높인다
