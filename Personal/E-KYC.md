#ekyc

## E-KYC 시스템을 구축하면서 만났던 문제들

1. 도메인에 대한 이해 부재 -> 왜 필요하고 무엇을 해결해야 하는지,
2. 프로세스에 대한 이해 부재 -> 직접 다양한 E-KYC 진행으로 해결
3. 오픈소스 딥러닝 프레임워크에 대한 이해 부재 -> 시간이 해결
4. 양질의 데이터 부족 -> 합성 데이터 생성 (Inpaint, SAM)


## 앞으로 남은 문제

- [ ] 합성 데이터로 훈련한 모델 검증
- [ ] 데이터 병합, 필드별 객체 탐지 모델 훈련 -> 개발 및 검증
- [ ] 주파수 기반 이상탐지 모델 훈련 (위조 신분증 검출)
