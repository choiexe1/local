## Stream
- 데이터의 흐름
- `StreamController`로 제어
- `Future`는 단발성 비동기 호출
- `Stream`은 관찰형 비동기

```dart
// StreamController 선언
final controller = StreamController<String>();

// 데이터 추가
controller.add('Hello');

// 데이터 구독
controller.stream.listen((data) {
	print(data); // Hello
})

// 구독 종료
controller.close();
```
