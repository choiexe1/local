
## 사용 금지
- `dynamic`
- `!`
- `late`

## 캡슐화
- private
	- 멤버 앞에 `_`를 붙이면 private 멤버가 됨
	- 같은 파일 내에서는 접근됨
- public
	- 기본 값
- 다트는 `protected` 키워드가 없다.
- `getter` 읽기 전용 속성 정의
- `setter` 쓰기 전용 속성 정의
- 개발자의 잘못된 접근에 대한 제어 방법 제공

## Collection

### List
- 다트는 Array가 없고 리스트만 존재함
- 따라서 동적으로 크기가 커질 수 있다.
- iterator 제공
### Set
- 중괄호로 Literal Set 선언
- iterator 제공
- Dart의 Set은 순서가 보장됨

### Map
- 키와 값의 쌍으로 이루어진 자료구조
- 값의 타입이 단일 타입이 아닌 경우 `dynamic` 사용
- 중괄호로 Literal Map 선언
