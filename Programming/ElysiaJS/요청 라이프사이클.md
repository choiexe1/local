#elysiaJS


```
요청
  ↓
onRequest        ← 가장 먼저 (로깅)
  ↓
derive()         ← 동기 컨텍스트 확장
  ↓
onParse          ← body 파싱
  ↓
onTransform      ← 검증 전 데이터 변환
  ↓
Validation       ← 스키마 검사 (params, body 등)
  ↓
resolve()        ← 비동기 컨텍스트 확장
  ↓
beforeHandle     ← 핸들러 전 (인증)
  ↓
핸들러
  ↓
afterHandle      ← 응답 수정
  ↓
mapResponse      ← 최종 응답 변환
  ↓
onResponse       ← 응답 후 (로깅)
  ↓
onError          ← 에러 발생 시
```
